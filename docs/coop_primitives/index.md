# Co-op primitives

Luminism’s practical path is to treat governance as an engineering problem: define simple primitives that real groups can adopt, measure, and iterate.

This section frames those primitives using **co-ops** as the grounding analogy.

- **Resource co-ops**: manage scarce resources with fair pricing + explicit congestion handling.
- **Membership & accountability**: resist Sybils and align incentives using dues/bonds and portable “member state”.
